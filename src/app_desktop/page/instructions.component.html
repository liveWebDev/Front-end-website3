IMAP - An introduction

IMAP and POP3 are protocols that allow you to download email messages from your mailbox and access them with Feasy app. Feasy can be configured on any standard IMAP email server using the IMAP and SMTP.

IMAP is more recent than POP and allows a two-way synchronization between the email server and your Feasy linked inbox. IMAP is recommended when you want to access the same account from multiple email clients.

Enable IMAP Access

To enable IMAP Access for your mail account:

  1. Log in to your Email account
  2. Go to Settings, and choose, POP/ IMAP and Email Forwarding.
  3. Under IMAP Access, select Enable.
  4. Configure Auto-Expunge and Folder View Settings, if it necessary.

IMAP and SMTP connection data

This information strongly depends on your email provider. You can find it on a settings page, or in a documentation.

Incoming Server Settings configuration example (IMAP):

  • Incoming Server Name: imap.yourmail.com
  • Port: 993
  • Require SSL: Yes

Outgoing Server Settings configuration example (SMTP):

  • Incoming Server Name: smtp.yourmail.com
  • Port: 465 with SSL or
  • Port: 587 with TLS

In order to interact correctly with such services we need to use standard authentication.

  • Login - usually coincides with email address you want to link
  • Password - your mailbox password
